## 定义
用户界面一部分，可以互相嵌套，有自己的样式，模版，逻辑

首字母大写的函数

组件定义使用：

```javascript
//定义
function Button() {
  return <button>click</button>
}
//使用
<Button/>
```

## useState（数据驱动试图，类似双向绑定，响应式）
react hook函数，允许向组件添加一个状态变量，从而控制影响组件的渲染效果

```javascript
//返回一个数组，第一个变量为状态变量，初始化为0，第二个变量为一个改变count的函数
const [count useCount]=useState(0)
```
