搭建博客的初衷

📌我搭建博客的原因：

某一天，为了解决某个问题，用搜索引擎去找办法，于是搜到了别人的博客里面的一篇文章，发现这个博客的样式很炫酷✨。然后就开始了我自己搭建博客的历程。

💡为什么要分享呢？

搭建完博客，总不能让它空有外壳吧，于是我开始每学一个新东西都开始记笔记（其实在这之前我从来都不写笔记，也不爱写东西💤），一步步把博客填充了一些内容。然后其实会发现能够更好的看清楚自己，也提高了一些写文档的能力。

所以我想，在这个信息爆炸的时代，我们每天都会接触到大量的信息，但这些信息往往在我们的脑海里停留不了多久。或许真正有价值的是能够把自己表达出来，比如刚学会了一个超级炫酷的新技能，或者灵光一闪解决了个很难的问题，甚至是看了一场电影后心里泛起的小波澜……而这些东西都可以通过写博客的方式记录下来。这些看起来不起眼的记录，随着时间的推移，会变得越来越珍贵。它们不仅能够帮助我更好地认识自己，或许也能够解决别人的某个问题。

hexo 是何物

**💡****相关文档**：[官方文档](https://hexo.io/zh-cn/docs/)

💡****环境：node.js 12.0 及以上版本、git

💡****基本介绍：hexo 是一个基于 Node.js 的快速、简洁且强大的静态博客框架，它通过将 Markdown 文章转换为静态网页（HTML、CSS、JS 等），帮助用户轻松搭建个人博客或文档站点，无需依赖数据库或后端服务器（更适合前端同学的体质🤸）。

📌创建项目三步曲

1️⃣安装 hexo ： npm install -g hexo-cli (全局安装 hexo)

2️⃣初始化项目：

- hexo init <folder> （创建项目，本质上就是 clone 了 hexo 官方的一个 git 仓库）
- cd <folder>
- npm i （安装依赖）

3️⃣启动项目：hexo server
